How to use this Product Manager

Step 1 — Add a product: Click "Add New Product" in the sidebar or the blue button. Fill in the form on the left side.

Step 2 — Live preview: The card preview on the right updates automatically as you type. You can see exactly how it will look on the store page.

Step 3 — Copy the code: Once you're happy with the preview, click "Copy code". This copies the HTML snippet for that product card.

Step 4 — Paste into index.html: Open index.html in any text editor (Notepad is fine). Find <!-- ADD NEW CARD HERE --> and paste right before it.

Step 5 — Add real images: Replace the colored placeholder in each .carousel-slide with: <img src="images/your-screenshot.jpg" alt="...">

Step 6 — Edit or delete: Go to "All Products", find the product and click ✏️ Edit or 🗑️ Delete. Your changes auto-save in this browser.

Important notes

⚠️ Products are now synced to GitHub Gist — accessible from any browser once set up.

📁 Both index.html and product-manager.html should be on the same domain.

🖼️ For real screenshots, put your image files inside an images/ folder next to index.html.

GitHub Gist sync

Your products are stored in a private GitHub Gist — accessible from any browser, any device. Set this up once and you're done.

GitHub → Settings → Developer settings → Personal access tokens → Classic → tick gist scope only
From your Gist URL: gist.github.com/yourusername/THIS_PART · filename must be products.json
Manual sync

Products auto-sync when you hit Go Live. Use these buttons if you need to manually push or pull.

Quick setup guide
  1. Go to github.com and sign in
  2. Click your avatar → Settings → Developer settings
  3. Click Personal access tokens → Tokens (classic)
  4. Click Generate new token (classic)
  5. Give it a name (e.g. "Lifehut Studio"), tick only gist, click Generate
  6. Copy the token (starts with ghp_) — paste it above
  7. Go to gist.github.com → New gist
  8. Filename: products.json, Content: []
  9. Click Create secret gist
  10. Copy the Gist ID from the URL → paste it above
  11. Click Save config then Test connection
